在创宇实习两个月的一些收获

发表于 2016-08-31

大三之前的这个暑假，我在知道创宇实习了两个月，收获颇丰。在公司里，每周末要写周报：总结本周工作内容、计划下周工作内容和随便说点啥。现在实习结束了，我也写个”两月报”，总结我实习过程中的收获、展望将来的计划。

最大的收获是和人的接触。实习的过程中，跟部门老大练哥和带我的大龙哥有数次长谈，每次都有”听君一席话，胜读十年书”之感。在跟他们的交流过程中，我了解到了一些公司的运作模式、对行业多了些认识、对自身的发展和定位有了许多思考。即便是简短的交流，也常常能打开眼界或是引发反思。谢谢你们！

在日常的工作过程中常向大家请教技术问题，雨哥教我的’127.0.0.1 和 localhost 的区别’、石航教给我的’python -m SimpleHTTPServer port’，等等，让我 get 到许多新知识。感谢大家！

其次的收获并不是具体的知识，而是我意识到了一些事情。这两个月，工作的主线是对蜜罐的调研和部署。中文互联网上，跟蜜罐相关的资料很少，所以我不得不大量使用 Google 、Stackoverflow 等网站查阅英文资料，跟外国博主发邮件交流，深切感到自己的英文读写能力还有很大不足。这两个月读下来的英文材料，肯定是超过我上大学来两年之和了。六级远远不是终点，互联网上的中文资料，只是互联网上宝贵财富的沧海一粟。

除了英文能力，我还意识到 Unix 系操作系统(Linux, macOS)的使用能力也很重要。去实习之前，我的简历上写着”会简单使用 Linux”，没想到使用 Linux 几乎成了最重要的技能，而且”简单使用”远远不够。但凡是开源软件，基本上极少有在 Windows 上能用的（如有也一般是跨平台通用的软件）。Linux，我还是得好好再学，我计划在新学期里每周抽出固定的时间，踏踏实实看一遍《私房菜》。Mac 的使用也是非常普遍的，macOS 是基于 FreeBSD 的操作系统，和 Linux 上的很多工具都兼容，非常方便，而且用起来确实好使。Windows 的问题，可以在 Mac 上用 Parallel Desktop 安装 Windows 虚拟机，很好用。据我观察，我们部门几乎所有人的工作用机都是 Mac，也有大牛在 Thinkpad 里装 Linux :)。不过具体情况还是具体分析，我们公司主要用 Python 开发，如果是用 .NET 的公司，估计还是得用 Windows 吧。

实习还带给我了一个收获，就是大大提高了安全意识。小时候看讲网络安全的书，说新装的 WinXP，如果不打补丁，插上网线后几分钟内就有可能中毒。我当时还不信，我就老老实实的用，不上乱七八糟网站、不下连七八糟的软件，怎么会中毒呢？现在发现这确实太容易中毒了。我也更换了我的所有主要网站账户和 APP 的密码，让他们相互之间没有重复，且基本都是”数字+大小写字母+特殊符号”，比之前密码的复杂度提高太多了。给大家分享一个建议：最好将自己的密码分级管理，比如第一级是最重要的密码(邮箱、微信、银行卡)，然后是次重要的，最后预备专门注册乱七八糟的小网站和 APP 的密码。不要将他们设置成一样的。

关于蜜罐也了解一些，但是也都是些表层的知识，甚至还算不上入门，暂且不总结了，还需要努力深入。通过蜜罐，我看到了一个更加广阔、五彩缤纷的世界，一个由于我的等级太低而无法踏入的世界，有点动成线、线动成面之感。持续修炼吧，把时间花在值得投入的事情上。白日莫闲度，青春不回头。

下一步的计划，除了补习 Linux，在学好大三上学期的专业课的基础上，也打算好好学习计算机网络的相关知识，计划读下来《TCP/IP详解》(不知道能不能读下来，加油)，并且好好读读《深入理解计算机系统》。上学真好，下学期一定要保持每周的运动量，形成习惯。恐怕是现在没有养成运动的习惯，工作后也很难了。

我在上完大二的时候，觉得自己也学了不少知识了，我想，我是不是已经由”不知道自己不知道”的阶段，慢慢成长为”知道自己不知道”呢？经过此次实习，我发现我还远远没有脱离”不知道自己不知道”：不知道的东西还太多，不知道自己不知道的东西也有太多。我想到了三句话：”哀吾生之须臾、羡长江之无穷。”、”怕什么真理无穷，进一步有进一步的欢喜。”、”路漫漫兮修其远，吾将上下而求索。”

谢谢陈仕玺，我是看到他分享的信息（黑哥的朋友圈），我才能争取到面试的机会；感谢保龙哥，你将我带入公司，我跟你学到了很多 PoC 相关的知识；感谢静静姐，希望我今后遇到的 HR 都像你一样和善。谢谢你们！
