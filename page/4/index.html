ssh 连接时出现 too many authentication failures error

发表于 2015-09-24
我在mac上，尝试用ssh连接我的ubuntu主机，但报错：1Received disconnect from x.x.x.x: Too many authentication failures for username我在这个网址找到了一个解决方案：1ssh -o PubkeyAuthentication=no username@x.x.x.x就可以正常输入密码并登陆了。似乎是~/.ssh中的密钥太多了，如果把这些密钥删除也能解决问题。
